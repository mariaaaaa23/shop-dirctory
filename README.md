# Shop Directory Project 🛒

این پروژه یک سیستم مدیریت آگهی و دایرکتوری فروشگاهی است که با استفاده از فریم‌ورک **Laravel** توسعه داده شده است. هدف از این پروژه، پیاده‌سازی یک پلتفرم منعطف برای مدیریت آگهی‌ها، دسته‌بندی‌ها و تعاملات کاربران می‌باشد.

## 🚀 ویژگی‌های کلیدی (Key Features)
*   **سیستم مدیریت کاربران:** احراز هویت امن و مدیریت پروفایل کاربری.
*   **مدیریت آگهی‌ها:** قابلیت ثبت، ویرایش و حذف آگهی‌ها توسط کاربران.
*   **سیستم دسته‌بندی (Categories):** دسته‌بندی هوشمند محصولات و آگهی‌ها.
*   **بخش علاقه‌مندی‌ها (Wishlist):** امکان ذخیره آگهی‌های مورد علاقه کاربران.
*   **پنل مدیریت:** مدیریت جامع بر روی محتوا و کاربران سایت.

## 🛠 تکنولوژی‌های استفاده شده
- **Backend:** Laravel 10 (PHP 8.2)
- **Database:** MySQL
- **Frontend:** Blade Templates / Bootstrap
- **Tools:** Git, Composer
